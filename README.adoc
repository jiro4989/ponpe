== Usage

とりあえず、pͪoͣnͬpͣoͥnͭpͣa͡iͥnする。

[source,bash]
----
% ponpe ponponpain haraita-i
pͪoͣnͬpͣoͥnͭpͣa͡iͥn
----

第2引数以降を全部1文字目に結合するので、以下のようなこともできる。

[source,bash]
----
% ponpe ____ dddd aaaa tttt eeee
_ͩͣͭͤ_ͩͣͭͤ_ͩͣͭͤ_ͩͣͭͤ
----

== Install

以下のコマンドを実行する。

[source,bash]
----
make install
----
